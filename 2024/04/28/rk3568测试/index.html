rk3568测试

Posted on 2024-04-28
  • Yume Shoka
Home _驱动子系统

一、版本升级

1、找到版本镜像文件

每次从仓上拉下新代码，编译之后就会生成该代码的版本，如果本地进行更改，编译之后会得到本地修改之后的版本
位置：out\rk3568\packages\phone\images

2、使用RKDevTool将镜像文件烧到板子中(在烧之前需先执行DriverInstall.exe驱动安装操作)

需将镜像文件和所烧文件一一对应

二、测试用例

1、找到测试用例所在

位置：drivers\peripheral\display\test\unittest\standard

2、在.gn文件中添加相应的测试用例

3、编译（需要进行指定编译）

输入：./build.sh –product-name rk3568 –build-target + 指定路径下的gn文件或者指定gn文件的简称
ege: ./build.sh –product-name rk3568 –build-target hdf_test_display

三、跑测试用例

1、查看板子是否连接成功

输入：hdc list targets

2、将测试文件推入板子

输入：hdc file send 测试文件 目标目录

3、进入板子，进行具体操作

输入：hdc shell

4、进入目标目录

输入：cd 目标目录

5、执行测试用例，查看结果

输入：./目标文件
