一、概述

本文主要描述在PVE7.4中启用NVIDIA RTX 2060显卡虚拟化（vGPU）。
硬件和软件的型号和版本信息：
主板：Supermicro X10Dri
CPU：Intel(R) Xeon(R) CPU E5-2680 v4 @ 2.40GHz
显卡：NVIDIA Corporation TU106 [GeForce RTX 2060 SUPER]
PVE：7.4-3

二、BIOS设置

显卡虚拟化需要开启以下BIOS设置：
VT(Intel Virtualization Technology)
VT-d(Intel Virtualization Technology for Directed I/O)

三、安装依赖的软件包

配置软件源

sed -i 's|^deb http://ftp.debian.org|deb https://mirrors.ustc.edu.cn|g' /etc/apt/sources.list
sed -i 's|^deb http://security.debian.org|deb https://mirrors.ustc.edu.cn/debian-security|g' /etc/apt/sources.list
source /etc/os-release
echo "deb https://mirrors.ustc.edu.cn/proxmox/debian/pve $VERSION_CODENAME pve-no-subscription" > /etc/apt/sources.list.d/pve-no-subscription.list

同步软件包列表

apt update

安装软件包

apt install -y build-essential dkms pve-headers-$(uname -r) mdevctl

四、修改系统设置

修改/etc/modules

echo vfio >> /etc/modules 
echo vfio_iommu_type1 >> /etc/modules 
echo vfio_pci >> /etc/modules 
echo vfio_virqfd >> /etc/modules

修改/etc/modprobe.d/pve-blacklist.conf

echo  "blacklist nouveau" >> /etc/modprobe.d/blacklist.conf

修改/etc/modprobe.d/kvm.conf

echo "options kvm ignore_msrs=1" > /etc/modprobe.d/kvm.conf

修改/etc/default/grub
将“intel_iommu=on iommu=pt”添加到GRUB_CMDLINE_LINUX_DEFAULT
GRUB_CMDLINE_LINUX_DEFAULT=”quiet intel_iommu=on iommu=pt”
刷新grub重启host

update-grub
update-initramfs -u -k all
reboot

五、host安装vGPU驱动

创建目录/opt/vgpu_unlock-rs/target/release

mkdir -p /opt/vgpu_unlock-rs/target/release

使用scp命令上传文件：NVIDIA-Linux-x86_64-525.85.07-vgpu-kvm-custom.run、libvgpu_unlock_rs.so
添加可执行权限

chmod +x NVIDIA-Linux-x86_64-525.85.07-vgpu-kvm-custom.run

安装驱动

./ NVIDIA-Linux-x86_64-525.85.07-vgpu-kvm-custom.run

提示是使用dkms，选择Yes
安装完成后，检查是否安装成功

nvidia-smi
mdevctl types

六、guest安装vGPU驱动

使用q35类型的虚拟机
向虚拟机添加设备
勾选Advanced和PCI-Express
开启虚拟机
Windows系统
安装驱动
执行：528.24_grid_win10_win11_server2019_server2022_dch_64bit_international.exe

七、验证

Windows系统中打开任务管理器，点击“性能”，可以看到GPU及其使用率
在PVE的Shell中执行nvidia-smi也可以看到vGPU

八、参考资料

在Proxmox VE 7.4 中开启NVIDIA P106显卡虚拟化(vGPU): https://foxi.buduanwang.vip/virtualization/pve/2815.html/
在Proxmox VE 7.1 中开启vGPU_unlock，实现显卡虚拟化: https://foxi.buduanwang.vip/virtualization/pve/1683.html/
