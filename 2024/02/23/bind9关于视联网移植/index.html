入视联网

extern int v2v_start(const char *mac);

v2v_drv 提供的入视联网接口
参数：入参，视联网的虚拟mac
函数作用：根据提供的视联网的虚拟mac去与自治云通信，实现此视联网mac的入网

extern int v2v_checkin_is_ok_or_not();

v2v_drv 提供的入视联网接口
参数：无参
函数作用：判断当前mac是否为入网状态，一般在收发视联包之前，都应调用此接口进行判断当前是否为入网状态

收发视联包

extern int v2v_checkin_is_ok_or_not();

v2v_drv 提供的入视联网接口
参数：无参
函数作用：判断当前mac是否为入网状态，一般在收发视联包之前，都应调用此接口进行判断当前是否为入网状态

extern int v2v_sendto(const char *dst_number, void *data, int len, int timeout);

v2v_drv 提供的入视联网接口
参数：
dst_number : 入参 目标地址的20位号码
data : 入参 发送数据
len ：入参 发送数据长度
timeout ： 入参 超时时间 默认 100ms

extern int v2v_recvfrom(char *remote_number, int remote_number_len, void *data, int len, int timeout);

v2v_drv 提供的入视联网接口
参数：
remote_number : 出参 源地址的20位号码
remote_number_len : 入参 源地址长度
data ：出参 收到视联包的数据
len ：入参 视联包的长度
timeout ： 入参 超时时间 默认 100ms
