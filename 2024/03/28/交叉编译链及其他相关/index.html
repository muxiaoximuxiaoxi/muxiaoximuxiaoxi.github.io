设置 Linux 编译链接库的步骤

1、拿到相应的连接包
2、解压到 /opt 目录下
3、设置 PATH 路径 具体操作如下：
vi /etc/profile
export PATH=”/opt/arm-hisiv200-linux/target/bin:$PATH”
source /etc/profile
4、通过 echo $PATH 查看PATH添加是否成功
5、在根目录下的 CMakeLists.txt 增加相应编译链接指令
ELSEIF (CROSS_COMPILE_PLATFORM MATCHES “QM2”)
SET(CMAKE_C_COMPILER arm-hisiv200-linux-gcc)
SET(CMAKE_CXX_COMPILER arm-hisiv200-linux-g++)
MESSAGE(“– Current Platform: ${CROSS_COMPILE_PLATFORM}”)
MESSAGE(“– Current QM2_CMAKE_C_COMPILER: ${CMAKE_C_COMPILER}”)
MESSAGE(“– Current QM2_CMAKE_CXX_COMPILER: ${CMAKE_CXX_COMPILER}”)
6、 Linux 可能需要额外安装一些包
yum -y install zlib.i686 zlib
如果还缺少某些包，直接网上搜就行，按照教程安装即可

设置 安卓 编译链接库的步骤

以 android-ndk-r10e-linux-x86_64.bin 为例
1、拿到相应的连接包
2、解压到 /opt 目录下
3、chmod 777 android-ndk-r10e-linux-x86_64.bin
4、./android-ndk-r10e-linux-x86_64.bin
得到 android-ndk-r10e 目录
5、设置 PATH 路径 具体操作如下：
vi /etc/profile
export PATH=”/opt/android-ndk-r10e:$PATH”
source /etc/profile
6、通过 echo $PATH 查看PATH添加是否成功
7、./make_all.sh -shared QM3 使用 sdk 编译脚本进行验证

sdk 新环境

1、安装 gcc g++ cmake
2、Ubuntu :
apt-get install build-essential
Centos :
yum groupinstall “Development Tools”

sdk demo 添加 ipv4 地址 指令

ifconfig vmbr1:0 192.168.3.99 netmask 255.255.255.0
ip addr add 192.168.2.99/24 dev ens18:0
